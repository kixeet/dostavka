### Введение 
Система доставки с возможностью оценки работы, времени выполнения и прочего

#### Примеры
- Конпка старт запускает
- Кнопка оценить выдает возможность оценки
- Кнопка поставить задачу устанавливает таймер
- Кнопка просмотр задачи показыавает времяч на выполнение
- Кнопка репутиации показывает репутацию
- Кнопкка оценки выдает оценки
- кнопка истьоия оценко выдает историю оценгко
- Кнопка назначить задачу создает задачу
- Кнопка сброс сбрасывает задачи

##### Тех требования
Работостпособность
Безотказность
Автономное функционирование
